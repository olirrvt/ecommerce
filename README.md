## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/olirrvt/ecommerce.git
```

Entre no diretório do projeto

```bash
  cd ecommerce
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor do Vite

```bash
  npm run dev
```
Acesse pelo seu navegador

```bash
http://localhost:5173/
```

## Stack utilizada

**Front-end:** Html5, Css3, ReactJS, Vite
